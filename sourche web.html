Apa itu Windows Insider?

Posted on June 26, 2015 by broxioux

Sampai saat ini, Microsoft masih bekerja untuk mengembangkan Windows 10 hingga pada 29 Juli 2015. Pada tanggal tersebut, Windows 10 telah menjamin baik penggunaannya dari segi tampilan hingga performanya. Namun, dalam pengembangannya, Windows 10 dikembangkan tidak hanya dari pihak dalam (Microsoft) saja, tetapi bantuan dari luar.

“Loh, maksudnya kolaborasi dengan perusahaan lain?”. Bukan. Bantuan dari luar yang dimaksud adalah semua orang. Semua orang yang ingin ikut membantu mengembangkan (tanpa paksaan) dengan memberikan feedback kepada tim pengembang.

Nama program tersebut adalah Windows Insider. Windows Insider merupakan salah satu cara bagaimana pengguna (disarankan untuk IT pro, tetapi semua orang boleh ikut) turut ambil bagian sehingga Windows 10 yang akan ada nantinya lebih personal, sesuai dengan keinginan dari banyak pengguna (Insider member). Windows Insider dipimpin oleh Gabriel Aul, seorang pemimpin program ini yang dijuluki sebagai Ring Master.

Program ini bisa dikatakan unik karena mengajak orang yang bukan dari bagian pekerja Microsoft/Windows untuk membantu pengembangan Windows 10 baik tampilan, performa maupun mengusung fitur baru pada Windows 10. Program sejenis ini sebelumnya sudah pernah dilakukan, dengan nama “Preview for Developers”, pada transisi dari Windows Phone 8 ke Windows Phone 8.1. Dan program ini pun bisa dikatakan berhasil, karena hingga sekarang Windows Phone 8.1 masih dapat bekerja dengan baik berkat campur tangan pengguna Windows Phone yang peduli. Berdasarkan pengalaman dari program ini juga, perusahaan-perusahaan lain pun mengikuti jejak Microsoft untuk melakukan Developers Preview (contohnya Google dan Apple).

Cara membantunya pun cukup mudah, hanya dengan memberikan feedback melalui app Windows Feedback, dan kemudian nanti akan diproses/dikerjakan. Sebagai Insider member (panggilan untuk orang yang ikut program Windows Insider) pun kita bisa membantu meski tanpa memberikan feedback. Hanya dengan upvote beberapa/semua feedback pilihan, kita sudah turut membantu pengembangan Windows 10.

Keuntungan menjadi Windows Insider adalah mendapatkan versi Windows 10 terbaru dengan label preview alias pratinjau sebelum dirilis, kita bisa dapat mencicipinya terlebih dahulu dibandingkan pengguna biasa. Ingat, sebagai Windows Insider, kita bisa menikmati fitur terbaru, tetapi lebih baik lagi jika kita turut berkontribusi bagi perkembangan Windows. Pada tanggal perilisan Windows 10 pun, Windows Insider dapat memilikinya secara gratis pula, tanpa reservasi apapun.

Berminat? Unduh saja file ISO yang telah disediakan oleh Windows team secara gratis! Tetapi ingat, diharapkan untuk berkontribusi dalam pengembangan Windows 10 ini sehingga Windows 10 nantinya lebih baik dan lebih personal.

Nantinya akan ada update untuk build berikutnya, dan cara memperolehnya pun sangat mudah seperti meng-update komputer secara biasa. Update-nya pun dibagi menjadi dua, yaitu fast ring dan slow ring (oleh karena itu Gabe Aul sering dipanggil Ring Master). Fast ring adalah update yang kita terima akan lebih cepat didapatkan tetapi belum bisa dikatakan terlalu stabil dalam penggunaannya; lalu slow ring adalah update yang kita terima akan lebih lama untuk mendapatkan build update tetapi diyakini lebih stabil dibandingkan fast ring.

Apa kamu cukup antusias dengan program ini? Kalian bisa tinggalkan pendapat kamu di kolom komentar di bawah ini.
